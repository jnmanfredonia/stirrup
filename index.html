Welcome to Stirrup

Connect your Google Sheet using an Apps Script web app. This gives the app full read and write access — no Google Cloud account needed.

1

Open your Google Sheet

Open the Google Sheet where you want to store your data. If you don't have one yet, create a new blank spreadsheet at sheets.google.com.

2

Open Apps Script

In your Google Sheet, click Extensions → Apps Script. This opens the script editor in a new tab.

3

Paste the script

Delete any existing code in the editor. Paste the full contents of the Code.gs file you downloaded. Then click the save icon (or press Cmd+S / Ctrl+S).

4

Deploy as a web app

Click Deploy → New deployment. Click the gear icon next to "Select type" and choose Web app. Set:

• Execute as: Me
• Who has access: Anyone

Click Deploy. You may be asked to authorize — click through and allow. Copy the Web app URL that appears.
