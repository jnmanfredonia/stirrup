Welcome to Stirrup

Connect your Google Sheet using an Apps Script web app — no Google Cloud account needed.

1

Open your Google Sheet

Open or create a blank Google Sheet at sheets.google.com.

2

Open Apps Script

Click Extensions → Apps Script. This opens the script editor in a new tab.

3

Paste the script

Delete any existing code. Paste the full contents of Code.gs and save (Cmd+S).

4

Deploy as a web app

Click Deploy → New deployment → gear icon → Web app.
Set: Execute as Me, access Anyone. Click Deploy and authorize. Copy the Web app URL.
